# Kanna-Example
Kanna 사용 예시입니다.
    
    프로젝트가 실행이 되지 않을 때 대처
    
        1.  xcode의 버전을 최신으로 유지해줍니다.
        2.  pod의 설치 유무를 확인합니다.
        3.  podfile.lock을 지우시고 다시 pod install해줍니다.
    
    https://www.rottentomatoes.com/top/bestofrt/?year=2020
